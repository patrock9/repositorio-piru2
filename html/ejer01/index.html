Preguntas HTML

  1. Dar un par de ejemplos de uso de etiquetas meta.
    La etiqueta meta se puede usar para dar metadata al navegador, como por ejemplo:
    1. name="author" indicando el autor de la pagina HTML.
    2. charset="xxx" indicando la codificación de caracteres del documento.
    3. name="keywords" para definir palabras clave para los navegadores de busqueda.
    4. name"description" define una description para la página web.

  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML
    A la etiqueta se le pueden aplicar atributos como lang para indicar el lenguaje del documento, atributo style o tambien atributo class para luego referenciarlo a auna clase en una hoja de estilos.

  3. Como se relaciona un header http con la etiqueta head de HTML?
    En un header http se envia información (metadatos) relativa a la codificación del mensaje TCP entre el cliente y el servidor como por ejemplo, la versión del navegador, el nombre del hots, los lenguajes que acepta el cliente, la versión del servidor, la política de manejo de cache o las codificaciones.
    En la etiqueta <head> del documento HTML se describen metadatos propios del documento como el titulo, enlaces a scripts y hojas de estilo. En este se pueden declarar etiquetas <meta> que en caso de que el servidor web no haya informado al navegador cierta información de un header HTTP la proporciona en reemplazo.

  4. Que es UTF8?
    UTF8 es un formato de codificación de caracteres, de los mas usados de la web.
    Es de longitud variable con un máximo de 4 bytes, lo cual permite 65.535 posiciones de caracteres. Sus siglas vienen de 8-bit Unicode Transformation Format.

  5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
    Para aplicar estilo se utiliza el atributo style="xxxx" y dado que es un atributo global se puede aplicar a cualquier etiqueta, aunque claramente cobrará mas relevancia en algunas mas que en otras.

  6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?
    En la etiqueta <a> mediante el atributo target="_blank" se le indica al navegador que el hipervinculo debe abrirse en una nueva pestaña.

  7. Explicar cuando usarían etiquetas semánticas y con que propósito.
    Las etiquetas semanticas se utilizan para dar estructura al documento HTML (header, nav, section, article, aside, footer) y permiten a los motores de busqueda indexar mejor la página web.

  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.
    Aplicaría atributos como border-color para cambiar el color de los bordes, background-color para cambiar el color de fondo de la celda o text-align para cambiar la alineación del texto que contiene la tabla.

  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
    La etiqueta body en un documento HTML indica al navegador que dentro de ella se encuentra el contenido a mostrarse en pantalla al usuario.
    El body de un mensaje HTTP poco tiene que ver con lo antes descripto, en esta parte del mensaje HTTP es donde viajas los datos, cualquiera sea el formato, entre cliente y servidor, puede ser datos ingresados en un campo de formulario, archivos enviados mediante un proceso de carga al servidor, la página HTML que devuelve el servidor o texto explicativo de algún estado de error..

  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?
    El atributo colspan y rowspan para columnas y filas respectivamente, son los que se encargan de agrupas celdas en una tabla y siempre se aplican a una etiqueta <td> (fila).
